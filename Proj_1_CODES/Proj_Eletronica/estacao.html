Comentários: O objetivo dessa prática foi construir uma estação meteorológica. Para isso foi utilizado quatro diferentes sensores, um de temperatura (LM35), um de pressão (BMP085), um de humidade (DHT22) e um de luminosidade (LDR). Para mostrar os resultados obtidos foi utilizado um display LCD 2X16. Para ler os códigos de cada sensor, assim como converter os sinais analógicos e digitais lidos por cada um, foi utilizado o arduino UNO. Por fim, foi confeccionada a nossa placa (PCB). O primeiro passo para a realização da estação meteorológica foi calibrar os sensores de temperatura (LM35) e o de humidade (BMP180). Após feita a calibração do sensor de temperatura (LM35), foi feita a calibração do sensor de umidade (DHT 22). Para realizar este processo de calibração o sensor foi colocado no topo de seis recipientes diferentes, para que não entrasse em contato diretamente com a água evitando assim molhar o sensor, em que cada uma continha uma certa quantidade de sal dissolvido em agua, e para cada solução fizemos duas medições em que o tempo de estabilização para cada medição foi de aproximadamente 20 minutos.
